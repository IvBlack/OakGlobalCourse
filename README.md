# OakGlobalCourse
Сниппеты кода и документации, погружающие в краеугольные и продвинутые аспекты языка java.
Задача проекта - собрать коллекцию-роадмапу, по которой можно изучать материал, не обращаясь к поиску в сети.

Целевая аудитория: 
Люди, желающие освоить язык Java на высоком уровне. 
Этот курс не подойдёт тем, кто хочет пройтись "по верхам", 
он рассчитан на действительно сложные задачи, которые полностью раскрывают 
предметную область.
