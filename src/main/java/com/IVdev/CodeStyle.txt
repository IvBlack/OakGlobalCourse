В Java есть такое понятие как Code Style - это правила написания кода, и, в целом, 
все Java программисты в мире пишут код по одним правилам, это помогает нам легче разбираться в чужом коде. 
Есть языки, которым не повезло так сильно, как нам, и там в каждой компании свой код стайл и 
приходится постоянно его изучать на каждой новой работе. 
Вам необходимо выучить и соблюдать всего одни правила, потому что они универсальны.
